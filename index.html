App Guide
How each tab and feature works
🏪 POS
Browse your active inventory in a 2-column grid. Toggle between All / Sealed / Graded / Singles. Search by name, short name, or set. Tap an item image to see details and market price. Tap Add to Cart to begin a sale. Only items with a sticker price and POS checkbox active are shown.
🛒 Cart
Review cart items, adjust quantities, apply per-item or cart-level discounts ($ or %). Tap Checkout to process payment via Cash, Zelle, Venmo, or CashApp. Split payments supported. Completed sales deduct from inventory and log to the Sales Log.
📦 Inventory
Full stock list with live quantities. Tap the thumbnail or name to view a full-size image. Adjust quantities with +/− or by typing directly. Quantities update automatically when sales are completed.
🤝 Buy/Trade
Purchase: Log items you are buying. Add multiple items before logging. Each transaction records name, qty, price paid, photo, and notes.

Trade: Select items you are giving (from inventory), add items you are receiving (with stated value), set your credit ($ or %). The app calculates balance due. Trade deducts given items from inventory immediately.
⚙️ Setup
Google Sheets Sync: Publish your Inventory sheet as CSV, paste the URL, tap Sync. Replaces inventory and loads sticker prices.

Import XLSX: Upload the template file directly from your device.

POS checkboxes: Check/uncheck items in Set Sticker Prices to control what appears on the POS screen — useful for limiting to your show inventory.

Export Sales: Access via the Sales or Revenue stat cards. Exports a CSV of all transactions. Export & Clear Log resets for a new session.

Export Activity: Exports all purchases and trades from the Buy/Trade tab.
⚡ Apps Script Setup
Copy and paste this code into your Apps Script editor (Extensions → Apps Script), then deploy as a Web App:

function doGet(e) { try { const action = (e && e.parameter && e.parameter.action) || 'getInventory'; const ss = SpreadsheetApp.getActiveSpreadsheet(); const sheet = ss.getSheetByName('Inventory'); if (!sheet) throw new Error('No sheet named Inventory found'); const data = sheet.getDataRange().getValues(); // Auto-detect header row by finding row containing product_name let headerIdx = -1; for (let i = 0; i < data.length; i++) { if (data[i].some(c => c.toString().toLowerCase().trim() === 'product_name')) { headerIdx = i; break; } } if (headerIdx === -1) throw new Error('Cannot find header row with product_name column'); const headers = data[headerIdx].map(h => h.toString().toLowerCase().trim()); const col = name => headers.indexOf(name); const rows = []; const priceMap = {}; // Skip header row + description row (headerIdx + 2), read all data rows for (let i = headerIdx + 2; i < data.length; i++) { const r = data[i]; const name = (r[col('product_name')] || '').toString().trim(); if (!name || name.toLowerCase().includes('full product name')) continue; const short = (r[col('short_name')] || name).toString().trim(); const sp = parseFloat(r[col('sticker_price')]); const type = (r[col('item_type')] || 'Sealed').toString().trim(); const urlRaw = (r[col('tcgplayer_url')] || '').toString().trim(); const imgUrl = (r[col('image_url')] || '').toString().trim(); const match = urlRaw.match(/\/product\/(\d+)/); const idBase = match ? match[1] : name.replace(/[^a-z0-9]/gi,'').slice(0,8).toLowerCase() + i; const id = (type === 'Graded' ? 'gr' : 'ev') + idBase; rows.push({ id, name, set: (r[col('set_name')] || '').toString().trim(), short, type, qty: parseInt(r[col('quantity')]) || 0, url: urlRaw, img: imgUrl, alt_url: (r[col('alt_url')] || '').toString().trim(), cardladder_url: (r[col('cardladder_url')] || '').toString().trim(), notes: (r[col('notes')] || '').toString().trim() }); if (!isNaN(sp) && sp > 0) priceMap[id] = sp.toString(); } if (rows.length === 0) throw new Error('No data rows found after header. Check your sheet has data below row ' + (headerIdx + 3)); const result = action === 'getPrices' ? { prices: Object.fromEntries(rows.map(r => [r.short, priceMap[r.id]]).filter(p => p[1])) } : { inventory: rows, prices: priceMap, count: rows.length }; return ContentService .createTextOutput(JSON.stringify(result)) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({ error: err.toString() })) .setMimeType(ContentService.MimeType.JSON); } }
After pasting: Deploy → New deployment → Web app → Execute as: Me → Access: Anyone → Deploy → copy URL → paste in Setup.
📊 Google Sheets Structure
Recommended workbook layout:

Inventory tab — master source for items + sticker prices. Publish as CSV and sync into the app.
Sales Log tab — future: app will write completed sales here.
Buy/Trade Log tab — future: app will write purchase/trade activity here.
Inventory Snapshot tab — future: end-of-day stock export.

Column order for Inventory tab: product_name, set_name, short_name, item_type, quantity, sticker_price, image_url, tcgplayer_url, alt_url, cardladder_url, notes
